Library Description


Built using C++ and then transpiled to JS and WASM the Cryptographio library makes use of some of the most begginer friendly encryption processes to help students learn the ins and outs of cryptography. The code can be called as a module in HTML files or run in js files using NODE. The customer can encrypt multiple different phrases at a time.


The library can be easily accessed and used in different formats
