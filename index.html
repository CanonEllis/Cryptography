Library Description


Built using C++ and then transpiled to JS and WASM the Cryptographio library makes use of some of the most begginer friendly encryption processes to help students learn the ins and outs of cryptography. The code can be called as a module in HTML files or run in js files using NODE. The customer can encrypt multiple different phrases at a time. The objective of this library was to make the concept of cryptography open to a wider audience by showing off some of the lower level versions that existed before some of the more modern techniques.


The library can be easily access and used in different formats. The code base can also be implemented in previously made websites, for example, class websites to teach about computer security.
